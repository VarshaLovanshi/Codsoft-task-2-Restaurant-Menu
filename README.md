# Codsoft-task-2-Restaurant-Menu

📌 Project Overview:

This project showcases a modern Restaurant Menu UI designed for a food delivery application named Feasto. The interface focuses on providing users with a smooth and visually appealing food ordering experience by combining clean layouts, attractive food cards, intuitive navigation, and easy access to menu details. The design emphasizes usability, consistency, and an engaging user experience while making food discovery and ordering simple and efficient.


✨ Features:

🍔 Modern restaurant menu interface

🔍 Search bar for quick food discovery

🍕 Category filters for easy browsing

🖼️ High-quality food images and product cards

📄 Detailed food information screens

⭐ Ratings and pricing display

❤️ Wishlist/Favourite option

🛒 Add to Cart functionality

🧭 Bottom navigation bar

📱 Mobile-first design with a clean and consistent layout


🛠️ Tools Used:
Figma, Material Icons, Food Images & Illustrations, Interactive Prototyping


🎯 Learning Outcomes:

Through this project, I gained practical experience in:
1. Designing mobile interfaces for food delivery applications
2. Creating visually appealing layouts with proper spacing and alignment
3. Applying UI/UX principles to improve navigation and usability
4. Maintaining consistency in typography, colors, and components
5. Building interactive prototypes to simulate user interactions
6. Improving design hierarchy for a better user experience
